# Interview Training — Сервис тренировочных собеседований

## Слоган
**Собеседования без обязательств**

---

## Суть сервиса

Сервис предоставляет две основные возможности:

### Для кандидатов
- Пройти тренировочное собеседование без обязательств
- Проверить свои навыки как специалиста
- Получить обратную связь от эксперта
- Подготовиться к реальным собеседованиям

### Для экспертов
- Провести тренировочное собеседование без обязательств
- Получить денежное вознаграждение за проведение собеседования
- Поделиться опытом и знаниями

---

## Роли пользователей

### 1. Неавторизованный пользователь
- Видит главную страницу сайта
- Может нажать кнопку "Войти" (логин)
- Может нажать кнопку "Зарегистрироваться" (создать нового пользователя)

### 2. Кандидат (авторизованный пользователь без выбранных ролей)
- Может посмотреть страницу пользователя(свою) и выбрать роль(кандидат, эксперт, либо обе)
- Может посмотреть всех экспертов (фото, краткое описание, полное описание, доступные экспертные области, доступное время)

### 3. Кандидат (авторизованный пользователь с ролью "Кандидат")
- Может посмотреть всех экспертов (фото, краткое описание, полное описание, доступные экспертные области, доступное время)
- Может выбрать нужного эксперта с нужными навыками
- Может назначить время собеседования
- Перед созвоном переводит фиксированную сумму
- Если не оплатил — ссылка на созвон не отображается
- При оплате получает подтверждение через уведомления

### 4. Эксперт (авторизованный пользователь с ролью "Эксперт")
- Имеет навыки (skills), по которым может проводить собеседования
- Каждый навык подтверждается отдельно через админа
- Устанавливает стоимость за собеседование по каждому навыку
- Может описать тему собеседования для каждого навыка
- Имеет рейтинг на основе отзывов кандидатов
- Может выбрать язык собеседования (русский, английский)
- Имеет приоритет отображения в списке

### 5. Администратор
- Может посмотреть всех пользователей, экспертов, кандидатов, админов (фото, краткое описание, полное описание, доступные экспертные области, доступное время)
- Подтверждает, что эксперт действительно эксперт в данной области
- Подтверждает факт оплаты кандидатом
- Подтверждает факт оплаты эксперту
- Добавляет ссылку на созвон (Яндекс.Телемост)
- Управляет пользователями и собеседованиями
- Может читать чаты между экспертами и кандидатами (вопрос на рассмотрении)

## Сценарии использования

### Сценарий 1: Регистрация
1. Пользователь переходит на сайт
2. Нажимает "Зарегистрироваться"
3. Вводит email и пароль
4. Email изначально не подтверждён
5. Аккаунт создан, но функционал ограничен

### Сценарий 2: Первый вход после регистрации
1. Пользователь авторизуется
2. Система показывает уведомление: "Подтвердите email"
3. Пользователь может видеть экспертов, но не может назначить собеседование
4. Пользователь нажимает кнопку подтверждения email
5. Получает письмо со ссылкой подтверждения
6. Переходит по ссылке — email подтверждён
7. После подтверждения email пользователь получает уведомления на почту

### Сценарий 3: Выбор роли
1. После подтверждения email пользователь выбирает роль:
   - Кандидат
   - Эксперт
   - Кандидат и Эксперт (обе роли одновременно)
2. Админом стать нельзя — роль назначается только администратором
3. Роль можно изменить позже в профиле

### Сценарий 4: Работа с профилем пользователя
1. Изначально профиль содержит только email и пароль
2. Пользователь может добавить:
   - Фото
   - Краткое описание себя
   - Подробное описание
   - Telegram (для уведомлений через бота)
   - Временную зону (по умолчанию Москва)
3. Пользователь может изменить роль:
   - Быть кандидатом, стать экспертом
   - Быть экспертом, стать кандидатом
   - Выбрать обе роли

### Сценарий 5: Заполнение информации кандидата
1. Кандидат указывает удобное время для прохождения интервью
2. Кандидат может выбрать навыки (skills), которые предпочитает
3. Выбранные навыки автоматически попадают в фильтр поиска экспертов
4. Если навыки не заполнены — фильтр пустой, значения по умолчанию

### Сценарий 6: Заполнение информации эксперта
1. Эксперт указывает удобное время для проведения интервью
2. Эксперт выбирает навыки (skills), которые знает
3. Каждый навык требует отдельного подтверждения через админа
4. Эксперт указывает стоимость за собеседование по каждому навыку
   - По умолчанию: 1 собеседование ~ 1 час = 5000 рублей
5. Эксперт может описать тему собеседования для каждого навыка (опционально)

### Сценарий 7: Назначение собеседования
1. Кандидат переходит на страницу поиска экспертов
2. Фильтрует экспертов по навыкам
3. Сортирует по рейтингу, дате регистрации
4. Выбирает эксперта, просматривает его профиль
5. Выбирает доступное время
6. Отправляет запрос на собеседование
7. Эксперт получает уведомление

### Сценарий 8: Подтверждение собеседования экспертом
1. Эксперт видит новое собеседование в списке
2. Может быстро подтвердить время
3. Может предложить другое время
4. Может отказаться с указанием причины
5. Переходит к странице информации о собеседовании

### Сценарий 9: Оплата собеседования
1. Кандидат переводит сумму по СБП
2. Админ подтверждает факт оплаты кандидатом
3. Кандидат получает уведомление об оплате
4. Админ добавляет ссылку на созвон (Яндекс.Телемост)
5. После проведения собеседования админ подтверждает оплату эксперту
6. Эксперт получает вознаграждение

### Сценарий 10: Проведение собеседования
1. В назначенное время кандидат и эксперт переходят по ссылке
2. Ссылка ведёт на Яндекс.Телемост
3. Проводится собеседование
4. Кандидат может оставить отзыв эксперту

## Функционал уведомлений

### Каналы уведомлений:

#### 1. Email уведомления
- Работает только после подтверждения email
- События для уведомлений:
  - Подтверждение email
  - Назначение собеседования
  - Подтверждение собеседования экспертом
  - Отказ от собеседования
  - Напоминание о собеседовании (за 1 час)
  - Подтверждение оплаты
  - Изменение времени собеседования

#### 2. Telegram уведомления (опционально)
- Пользователь указывает Telegram username
- Подключается через бота
- Те же события, что и для email
- Мгновенная доставка

#### 3. SignalR уведомления (на сайте)
- Работает только при нахождении на сайте
- Всплывающие уведомления (toast)
- События:
  - Подтверждение оплаты
  - Новое сообщение в чате
  - Изменение статуса собеседования
  - Новое собеседование (для эксперта)

## Приоритет и рейтинг экспертов

### Рейтинг эксперта:
- Формируется на основе отзывов кандидатов
- Шкала: 1-5 звёзд
- Влияет на сортировку по умолчанию

### Приоритет эксперта:
- Изначально одинаковый у всех
- Новые эксперты отображаются выше (по умолчанию)
- Возможность платного поднятия (на рассмотрении)

### Сортировка для кандидата:
- По рейтингу (возрастание/убывание)
- По дате регистрации (новые/старые)
- По приоритету (если реализовано)

## Оплата и монетизация

### Для кандидата:
- Сумма зависит от навыка и эксперта
- Устанавливается при подтверждении навыка эксперта админом
- Может быть изменена только админом
- Оплата в рублях по СБП

### Для эксперта:

## Статусы собеседования

| Статус           | Описание | Доступные действия |
|-----------------------------|----------|-------------------|
| Запрос кандидата                                     | Запрос создан кандидатом | Эксперт: подтвердить/отказаться/предложить другое время |
| Подтверждено экспертом(Ожидает оплаты от кандидата)  | Эксперт подтвердил время | Кандидат: оплатить |
| Оплачено кандидатом                                  | Кандидат оплатил | Админ: добавить ссылку на созвон |
| Готово к проведению(Ссылка добавлена)                | Готово к проведению | Ожидание проведения |
| Проведено                                            | Собеседование состоялось | Кандидат: оставить отзыв |
| Завершено                                            | Отзыв оставлен | Нет действий |
| Отменено кандидатом                                  | Отменено кандидатом | Нет действий |
| Отменено экспертом                                   | Отменено кандидатом | Нет действий |
| Оплачено эксперту(отдельный признак)                 | Оплачено эксперту | Нет действий(выставляет админ) |
