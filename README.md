# interview-training
Сервис, который даст вам возможность пройти тренировочное собеседование, без обязательств. Платно
Сервис, который даст вам возможность провести тренировочное собеседование, без обязательств. Вы получите деньги

Адреса сайта:
interview-training.com
interview-training.ru
тренировочные-собеседования.ру

Основные тезисы:
Есть авторизированные пользователи и не авторизированные
Авторизированные: кандидат, эксперт, админ
Кандидат видит список экспертов. И может назначить с ними тренировочное интервью. Цена зависит от времени или фиксированная за интервью
Эксперт видит те интервью, что ему назначил кандидат. Назначает свободное у него время. Изначально навыки у эксперта на заполнены. При заполнении навыков идёт проверка админом.
Админ подтверждает навыки у эксперта 
Админ подтверждает факт оплаты кандидатом его интервью
Админ подтверждает факт оплаты экспертом его интервью
Есть русский и английский текст
